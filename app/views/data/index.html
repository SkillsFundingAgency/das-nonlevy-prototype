{{serviceName}} Analytics and dashboards

What is Analytics and dashboard work?

The analytics team handle all the data, analytics and visualisation for the service. This project created the design styles and sketches for future dashboards.

This documents the work to create design patterns for the dashboards and sketches and work to implement those.
